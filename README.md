# GNS Landing Theme

Tema personalizado y optimizado para Landing Page de GNS Group Advisory.

## Características

- **100% Limpio y Optimizado**: Sin código innecesario
- **Performance First**: Diseñado para máxima velocidad
- **Responsive**: Compatible con todos los dispositivos
- **SEO Ready**: Optimizado para motores de búsqueda
- **Fácil de Personalizar**: Código claro y bien comentado

## Estructura del Tema

```
gns-landing/
├── assets/
│   ├── css/          # Estilos adicionales
│   ├── js/           # JavaScript personalizado
│   └── images/       # Imágenes del tema
├── style.css         # Hoja de estilos principal
├── functions.php     # Funciones del tema
├── index.php         # Template principal
├── header.php        # Cabecera
├── footer.php        # Pie de página
└── README.md         # Este archivo
```

## Instalación

1. Subir el tema a `wp-content/themes/gns-landing/`
2. Activar el tema desde Apariencia > Temas en el admin de WordPress
3. Configurar el logo y menús desde Apariencia > Personalizar

## Optimizaciones Incluidas

- ✅ Emojis desactivados (mejor performance)
- ✅ Scripts innecesarios de WP removidos
- ✅ HTML5 Support
- ✅ Custom Logo Support
- ✅ Navigation Menus
- ✅ Widget Areas
